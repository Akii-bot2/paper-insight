# Paper Insight 📄

最新の学術研究をわかりやすく。毎日の発見を、あなたに。
Paper Insight は、複雑な論文を「3つのポイント」と「要約」で噛み砕いて紹介するWebメディアです。

## 🌐 サイトを見る
スマートフォン、PCどちらからでもアクセスできます。

**👉 [Paper Insight を開く](https://akii-bot2.github.io/paper-insight/)**

<br>

---

## 🛠 Features
- **Library**: 気になる記事をブックマーク、お気に入りの記事を「いいね」して保存。
- **Search**: キーワードや気分（Mood）で論文を検索。
- **Admin**: 専用の管理画面から記事データを簡単に作成（`?admin=true`）。

## 📱 Mobile Support
レスポンシブデザインに対応しており、スマートフォンでも快適に閲覧可能です。
